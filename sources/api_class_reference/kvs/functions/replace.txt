replace

Description
---------------------------------------------------------------------------------------------------------
.. list-table:: 

  * - boolean **replace** ( string *$key*, mixed *value*\[, int *$expiration* = 0\])

**replace** replace value

Parameters
---------------------------------------------------------------------------------------------------------
* **key**

  * key string

* **value**

  * value to be stored

* **expiration**

  * lifetime of the data

Return Values
---------------------------------------------------------------------------------------------------------
	**boolean** : return true if success, false is failed
