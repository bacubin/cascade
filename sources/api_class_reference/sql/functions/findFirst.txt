findFirst

Data is retrieved by the specified key

---------------------------------------------------------------------------------------------------------
.. list-table:: 

  * - mixed **findFirst** (string *$stmt_name*, array *$params*\[, mixed *$hint* = ``null``, boolean *$use_master* = ``false``\])

**findFirst** returns results of data from specified key

Parameters
---------------------------------------------------------------------------------------------------------
* **stmt_name**

  * name of statement

* **params**

  * parameters of condition

* **hint**

  * hint of FarmSelect

* **use_master**
