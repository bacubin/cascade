Cascade_DB_SQL_DataFormat::getMasterDSN
=========================================================================================================
Get master DSN

Description
---------------------------------------------------------------------------------------------------------
.. list-table::

 * - string **getMasterDSN** ( Cascade_DB_Criteria *$criteria* )

Get master server's DSN. If you want to get dynamically DSN will be determined based on the extraction conditions.

Parameters
---------------------------------------------------------------------------------------------------------
* **criteria**

 * Extraction condition

Return Values
---------------------------------------------------------------------------------------------------------
* **string**

 * Master server's DSN
